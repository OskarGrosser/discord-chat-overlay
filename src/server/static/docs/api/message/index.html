Message API

The Message API works in combination with the Chat Frontend. Requests sent to this API will be forwarded to /chat, where they will be handled appropriately.

Table of content
  1. New messages
  2. Edits to messages
  3. Deletions of messages

For all requests to this API, the following must be:

  • Method: POST
  • Content-Type: application/json
  • Body: 
    {
  id: number;
  type: "new"|"edit"|"delete";
  content?: any;
}

The sent type determines the content, if at all. More on the content in the following chapters.
